# atm-withdrawal-refill-optimization
Data-driven ATM cash demand forecasting and refill optimization with Random Forest models and Power BI dashboards.

 : This project analyzes ATM cash withdrawal patterns to understand cash demand, predict refill requirements, and optimize ATM replenishment schedules. The study uses historical ATM transaction data, machine learning models, and Power BI visualizations to support data-driven ATM cash management.

Objectives- 
Analyze ATM withdrawal trends.
Predict ATM cash withdrawal amounts.
Predict refill requirements using machine learning.
Improve ATM cash availability and operational efficiency.
Propose customer-focused service improvements for women and senior citizens.

Tools Used- 
Python (Pandas, Scikit-learn, Matplotlib)
Microsoft Excel
Microsoft Power BI
Analysis Performed
Data Cleaning and Preparation
Exploratory Data Analysis (EDA)
Random Forest Classification for Refill Prediction
Random Forest Regression for Cash Withdrawal Prediction
Dashboard Development in Power BI

Key Results-
Refill prediction accuracy: 92.5%
Cash withdrawal prediction R² score: 0.8292
Identification of ATM cash usage trends and refill patterns
Recommendations for ATM service improvements and future innovations

Future Scope-
AI-based refill optimization
Real-time ATM monitoring dashboards
Smart ATM service enhancements for women and senior citizens
Route optimization for cash replenishment operations
